General information and announcements

Announcements:

- My apologies about the site going through so many changes recently, ive been trying to (hopefully) finalize the modpack. A major change that I am looking at undertaking is the switch to a more recent multiplayer mod: Luna MultiPlayer (LMP). At this point, the old multiplayer client the server was using (Dark MultiPlayer/DMP) was just being recompiled for new versions of KSP- It was no longer being actively worked on. LMP is new (relatively) on the scene and adds many new features that DMP lacked, such as UDP network messages and stuff. if you so wish, you can check out the release thread here. Just be aware that as this is very new, it is seeing a lot of active development and so a lot of changes are happening very regularly. (posted 12/11/2017)
