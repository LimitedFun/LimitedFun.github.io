If you feel like you don't have enough mods, here are some other useful ones.

MODS:

>>>Unless otherwise specified, Get the primary zip archive from the link and place the contents of its ../GameData folder into your ../GameData folder.

 

>>>NOTE: DO NOT COPY OVER ANY INCLUDED COPIES OF ModuleManager.DLL BEYOND THE ONE THAT YOU MANUALLY INSTALL BELOW. IF YOU DO, YOU WILL HAVE ISSUES.

 

>>>MAKE SURE YOU READ THE DESCRIPTION AND INSTRUCTIONS SO THAT YOU KNOW ANY SPECIFIC INSTRUCTIONS FOR EACH MOD.

 
All Y'All Continued
- Adds a context menu to parts like solar pannels or radiators so you can activate all of a specific type of part on a ship from just one click (as opposed to having to select and activate each part individually.
 
Better Burn Time
- Gives a more accurate prediction of how long you will need to burn to complete a specified maneuver, as well as giving you a countdown timer that shows you when you should start burning in order to complete a maneuver.
 
Better Timewarp Continued
- Adds more timewarp options for faster rates (Useful for travelling to the additional planets added in the core modpack in a reasonable amount of time).
 
Correct CoL
- Adds a modified Center of Lift calculation to the Spaceplane Hangar to help with building more controllable aircraft.
 
Kerbal Alarm Clock
- Enables you to create alarms for stuff like transfer windows between planets, future maneuvers, etc. Interfaces with Precise Maneuver and Transfer Window Planner.
 
Kerbal Engineer
- Adds readouts for various bits of information such as Delta-V (dV, Î”V) or Thrust to Weight Ratio (TWR).
- Get the primary zip archive from the link above, and place the ../KerbalEngineer folder into your ../GameData folder.
- In order to ensure compatability, remove the parts folder within ../KerbalEngineer
 
KSP Addon Version Checker (AVC)
- Provides automated version checking, to make sure that the latest version of a mod is installed.
- Get the primary zip archive from the link above, and place the ../KSP-AVC folder into your ../GameData folder.
 
Precise Maneuver
- Adds an interface for more accurately planning maneuvers. Interfaces with Kerbal Alarm Clock.
- Place the contained ../PreciseManeuver folder into your ../GameData directory.
 
RCS Build Aid
- Helps you plan out the placement of RCS thrusters or engines by displaying the rotational torque that misaligned thrusters will produce.
- Grab the most recent release from the provided link.
 
Tac Fuel Balancer
- Enables you to rebalance or dump fuel on the go during any flight.
 
Take Command Continued
- Allows you to place Kerbals into an EVA seat as if it were a capsule, so you can populate them before a test or launch.
 
The Janitor's Closet
- Adds an editor tool to help sort or hide parts in the VAB or SPH.
 
Trajectories
- Displays more accurate atmospheric trajectories, so that you can more accurately land craft on atmospheric planets or moons.
 
Transfer Window Planner
- Enables you to figure out optimal times to launch to given planetary bodies from given planets or moons. Interfaces with Kerbal Alarm Clock.
